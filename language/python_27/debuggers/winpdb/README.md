# Background
- Winpdb is the graphical front-end to rpdb2. I'm interested in rpdb2
# Installation
- $ `python -m pip install winpdb-reborn` or the like
- $ `pip install wxPython`
    - wxPython is a requirement for winpdb, but NOT for rpdb2
# Usage
- $ `python -m rpdb2 my_program.py` or $ `rpdb2 my_program.py`
